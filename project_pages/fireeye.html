// FireEye //

FireEye is a websocket wrapper to send arbitrary data over a TCP/IP port from Python to Node.js. It was written specifically to stream video from a Raspberry Pi to a remote web server, but can be applied to any system capable of running python and connecting to a network.
To read full documentation here is a link to the GitHub source. This package is also hosted on PyPi and on NPM (Node Package Manager).
