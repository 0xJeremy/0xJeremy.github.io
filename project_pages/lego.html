// LEGO robots //

The LEGO robots shown below were created for ME-84, an undergraduate course on Robotics and Mechatronics. The programming for all these robots was done in MatLab.

These LEGO arms are IOT enabled. The arms are programmed using the sine and cosine function to draw oscillation patterns on a whiteboard table. On a web dashboard a user can enter a value between 0 and 100 to change the magnitude of the functions. The two arms are linked via Bluetooth, with arm also reading values from the web dashboard. This was accomplished by "hacking" the sensor port on the LEGO EV3 microcontroller to read values from a separate, web connected microcontroller (programmed in Python).

These "clocks" are inspiried by the Weasley clock in Harry Potter. The user can set their location on one clock, and that "location" will be sent to the other clock via Bluetooth, and displayed there. Both clocks are bidirectional and have 6 discrete location settings.

These LEGO towers were created to mimic a game of rock, paper, scissors. The user can enter either "Attack", "Defend", or "Plan" on a control panel, and their partner does likewise. The tower of the loser will then "fall" a portion of the way down. The first tower to fall completely is the loser of the game.

This LEGO gear train was meant to mimic an art installation at the MIT museum of science. The speed of the motor can be adjusted by the two buttons on the front. The gear ratio of this train is approximately 1:2,100,000,000.
