// undergraduate research project //

Over the summer of 2018, and into the fall semester I was working on an independent undergraduate research project in the Tufts ASAR Lab. The ASAR lab (link here) is the Automated Systems and Robotics Lab at Tufts University. Under my adviser Dr. Jason Rife, and PI in the ASAR lab, I worked on developing a method of control for a single user to fly multiple quadrotor drones.

To accomplish this, there were multiple pieces that had to work together. Firstly, I used JavaScript to fly the drone around the lab while issuing commands from my computer (the drone used in this research is the Parrot AR 2.0: link here). Next, I integrated a three dimensional positioning system that would provide feedback to my controller (the system used was the Marvelmind indoor navigation system: link here). The next step was to integrate the Python positioning system and the JavaScript flight control into a cohesive unit. This was accomplished using a local data socket. The last step for control over a single drone was to add data smoothing across the incoming position data, implement a proportional controller, and allow the user to enter desired locations for the quadrotor.

During the fall semester, the next steps will be to extend this control to multiple quadrotors using the same controllers and systems already in place for a single drone. Because of the semi-autonomous nature of the system, the user will not have to exert direct control over each drone at all times, and instead can specify tasks that will be carried out automatically. The full code for this project to date can be viewed on my Github, and is linked here.
